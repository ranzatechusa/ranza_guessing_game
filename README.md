# Ranza Guessing Game 🎮

A sleek, command-line number guessing game built in Rust. This is a featured project under the **Ranza Tech** venture.

[![Crates.io](https://shields.io)](https://crates.io)
[![License](https://shields.io)](#license)

## Overview

The computer picks a secret number, and it's your job to guess it! The game provides feedback on whether your guess is too high or too low, helping you narrow it down.

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
ranza_guessing_game = "0.1.3"
```

Or install the binary directly:

```bash
cargo install ranza_guessing_game
```

## How to Play

Run the game using:

```bash
cargo run
```

Follow the on-screen prompts to enter your guesses.
